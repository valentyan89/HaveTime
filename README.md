# Разработка бизнес-логики и файловой составляющей мобильного приложения "Умный календарь" для операционной системы Андроид на языке Котлин

## Описание проекта
Умный календарь — это мобильное приложение для операционной системы Android, разработанное на языке Kotlin, предназначенное для удобного планирования задач, событий и напоминаний.
Приложение помогает пользователю эффективно управлять своим временем, планировать ежедневные дела и получать уведомления о важных событиях. Основная цель проекта — реализовать удобный интерфейс и продуманную бизнес-логику для работы с календарными событиями.
В приложении реализована система хранения и обработки данных о задачах, возможность добавления, редактирования и удаления событий, а также механизм напоминаний.

## Основной функционал
- 📅 Просмотр календаря по дням
- ➕ Добавление новых событий
- ✏️ Редактирование существующих задач
- ❌ Удаление событий
- ⏰ Установка напоминаний
- 📋 Просмотр списка запланированных дел
- 🔎 Удобная навигация по датам

## Технологии
### В проекте используются следующие технологии:
- Kotlin — основной язык разработки
- Android SDK
- Android Studio
- Git / GitHub — система контроля версий
